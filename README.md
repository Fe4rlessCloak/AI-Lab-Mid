# 🧠 Tic Tac Toe AI (Minimax Algorithm)

This project implements an **unbeatable Tic Tac Toe AI** using the **Minimax Algorithm**.  
The goal is to create a computer player that **never loses**, regardless of the opponent’s moves.

---

## 🎯 Objective

- Build a Tic Tac Toe game
- Implement the **Minimax algorithm**
- Ensure the AI always plays optimally
- The AI should either:
  - ✅ Win
  - 🤝 Or force a draw  
  - ❌ Never lose
